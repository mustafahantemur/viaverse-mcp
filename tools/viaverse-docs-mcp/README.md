# viaverse-docs-mcp

This MCP server searches and reads the original detailed Viaverse markdown files.

## Setup

```bash
cd tools/viaverse-docs-mcp
npm install
npm run build
```

## Development Run

```powershell
$env:VIAVERSE_REPO_ROOT="C:\Projects\Viaverse\viaverse-platform"
npm run dev
```

## Tools

```text
search_docs
read_doc
list_docs
get_related_umls
get_context_pack
pre_coding_brief
check_forbidden_terms
```
